# Python Ofensivo

Esta carpeta contiene scripts de Python y apuntes relacionados con el aprendizaje de Python orientado a ciberseguridad (Python Ofensivo), basados en el curso de S4vitar.

## Contenido

### Conceptos Fundamentales
*   **[conceptos.md](conceptos.md)**: Notas básicas sobre intérpretes (python2 vs python3), pwntools, y ejecución desde terminal o shebang.
*   **[coordenada.py](coordenada.py)** / **[global_variable.py](global_variable.py)**: Ejemplos de uso de clases, métodos y alcance de variables.

### Estructuras de Datos
*   **Listas**: 
    *   **[listas.py](listas.py)**: Manipulación de listas, métodos como `.append()`, `.reverse()`, y bucles `for` y `while`.
    *   **[list_hackers.py](list_hackers,py)** / **[compression_list.py](compression_list.py)**: Operaciones avanzadas con listas o comprensiones.
*   **Tuplas**: **[tuplas1.py](tuplas1.py)**, **[tuplas2.py](tuplas2.py)**, **[tuplas3.py](tuplas3.py)**.
*   **Sets**: **[set1.py](set1.py)**, **[set2.py](set2.py)**.

### Funciones y Control de Flujo
*   **[functions.py](functions.py)**: Definición básica de funciones (`def`).
*   **[lambda.py](lambda.py)**: Uso de funciones lambda y `map()`.
*   **[conditional.py](conditional.py)**: Estructuras condicionales (`if`, `else`).

### Manejo de Errores
*   **[exceptions.py](exceptions.py)**: Bloques `try`, `except`, `else`, `finally` para manejo de excepciones (ZeroDivisionError, TypeError).
*   **[raise.py](raise.py)**: Cómo lanzar excepciones personalizadas.

### Otros
*   **[format.py](format.py)** / **[str_format.py](str_format.py)**: Formateo de cadenas.
*   **[example.py](example.py)**, **[test.py](test.py)**: Scripts de prueba y ejemplos varios.
